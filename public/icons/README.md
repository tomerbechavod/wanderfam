# PWA Icons

Replace the SVG files here with real PNG icons for production.

## Quick way to generate PNG icons:

1. Go to https://realfavicongenerator.net
2. Upload a 512×512 image (use the mountain emoji on a #2a6049 background)
3. Download the package and copy icons here

## Required sizes:
- icon-72x72.png
- icon-96x96.png  
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png  ← most important (Android)
- icon-384x384.png
- icon-512x512.png
- apple-touch-icon.png (180×180)  ← required for iOS
